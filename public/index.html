What it does

  • Fetches docs and removes non-implementation noise.
  • Keeps critical API anchors (method/path/header/pagination/signature).
  • Returns compact context packs for coding assistants and MCP clients.

Jobs To Be Done

  • Pull only task-relevant docs context for implementation.
  • Preserve exact API contract details needed to ship code.
  • Reduce token spend without sacrificing implementation quality.

Quick Install

git clone https://github.com/dhruvtoprani/docflow-mcp.git
cd docflow-mcp
npm install
npm run build

Run locally (stdio MCP)

npm run dev

Run as HTTP MCP server

npm run dev:http

Endpoint: http://127.0.0.1:3000/mcp

Health: http://127.0.0.1:3000/healthz

Connect to ChatGPT

  1. This server is live on Vercel.
  2. Use https://docflow-mcp.vercel.app/mcp as your connector URL.
  3. In ChatGPT, go to Settings -> Connectors -> Create.
