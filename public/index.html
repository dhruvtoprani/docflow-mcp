Quick start

git clone https://github.com/dhruvtoprani/docflow-mcp.git
cd docflow-mcp
npm install
npm run build
npm run dev:http

HTTP endpoint: http://127.0.0.1:3000/mcp and clip endpoint: http://127.0.0.1:3000/clip
