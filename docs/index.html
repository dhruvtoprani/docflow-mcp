Quick install

git clone https://github.com/dhruvtoprani/docflow-mcp.git
cd docflow-mcp
npm install
npm run build

Run locally

npm run dev

or run HTTP MCP server:

npm run dev:http

Endpoint: http://127.0.0.1:3000/mcp
