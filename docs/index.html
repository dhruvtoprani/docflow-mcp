Open Source MCP Tool

DocFlow MCP

Turn long API docs into compact, implementation-ready context packs for ChatGPT, Claude, Cursor, and other MCP clients.

View on GitHub Read README

What it does

  • Fetches a docs URL
  • Removes noise and converts useful content to markdown
  • Detects sections like auth, endpoints, params, errors, and rate limits
  • Returns a compact context pack for coding assistants

Quick install

git clone https://github.com/dhruvtoprani/docflow-mcp.git
cd docflow-mcp
npm install
npm run build

Run locally (stdio MCP)

npm run dev

Run as HTTP MCP server

npm run dev:http

Endpoint: http://127.0.0.1:3000/mcp

Health: http://127.0.0.1:3000/healthz

Connect to ChatGPT

  1. Deploy this server so it has a public HTTPS URL.
  2. Use https://your-domain.com/mcp as your connector URL.
  3. In ChatGPT, go to Settings -> Connectors -> Create.
